# 4BitCPU

> A simulated toy 4 bit CPU developed in Logisim Evolution v4.1.0 using basic logic gates, controlled buffer, input output button.



---

## Project Overview:

### Name: 4 Bit CPU
A custom 4-bit CPU built from logic gates, controlled buffer, and input output buttons.
### Purpose: Experimental
### Key Features
- 4 bit data path
- 4-bit opcode ISA (16 instruction max)
- Real time assembly programming and execution

## Architecture Summary:

- FSM (Finite State Machine) that tracks Fetch, Decode, Execute, and Writeback states
- Program Counter that stores the address of the next instruction to fetch from the ROM
- 16 words 8 bit wide ROM that stores program to be executed. Structure: [4 Bit Opcode] [4 Bit Operand]
- Control Unit that executes instruction and generate necessary control signal (i.e. access RAM, conditional jump etc.)
- ALU (Arithmetic Logic Unit) that performs Addition, Subtraction, Increment, Decrement, Bit Inversion, Bit Masking
- Registers used for temporary data storage
  - 2 Program Counter Registers (store next ROM address and current ROM address)
  - 2 Instruction Registers (store Opcode and Operand) 
  - 1 Accumulator Register (stores ALU result)
  - 1 Flag Register (stores flags i.e. Zero, Carry, Negative, Overflow generated by ALU)
  - 1 Memory Register (stores RAM address for read or write operations)
- 16 words 4bit wide RAM.
- Tiny assembler that translates Human Readable instruction into Machine Code.

## 4 Bit CPU Instruction Set Architecture (ISA)

### Overview

The ISA defines the machine instruction for a custom 4 bit CPU.

### Properties
- Opcode width: 4 bits
- Maximum Instruction: 16
- Implemented Instruction: 15
- Instruction Classes
  - Arithmetic
  - Logic
  - Program Control
  - Memory Access
